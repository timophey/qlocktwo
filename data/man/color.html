  • Отображение

  • Таймеры основные

    • ЧМ: задержка между печатью значений часов и минут
    • Печати: задержка между печатью каждого символа

  • Печать...

    • use HSV: применять модель HSV (Candle работает иначе)
    • random Hue: если выбрана модель HSV, берёт случайный оттенок с сохранением значений яркости и насыщенности
    • Leading One: ведущий пиксель
    • Tail clear: очищать экран последовательно, паравозиком

  • Функция Candle:

    • use Candle: активировать мерцание пикселей
    • Candle dx max: Максимальное смещение значения яркости при мерцании
    • Candle dx step: шаг смещения за один проход
    • Candle timer: таймер функции
    • delta < Value: падение яркости
    • delta > Value: повышение яркости
    • fall to 0: уронить значение на нулевую отметку
    • fall to 255: уронить значение на максимальную величину
    • bounce: менять направление функции при достижении крайних значений (0 или 255)
